# Introdução
Repositório para portfólio de projetos de Data Science e Machine Learning.

# Estrutura
O repositório está dividido em uma estrutura de diretórios para comportar os datasets, notebooks e scripts específicos de cada projeto:

|- Projeto 1
  |-- Datasets
  |-- Notebooks
  |-- Scripts
  
|- Projeto 2
  |-- Datasets
  |-- Notebooks
  |-- Scripts
  
|- Projeto 3
  |-- Datasets
  |-- Notebooks
  |-- Scripts

  # Acompanhamento
  Os projetos são em sua maioria de origem a partir dos cursos da Data Science Academy. Mas também há projetos independentes feitos para a prática dos
  conceitos aprendidos no curso.
